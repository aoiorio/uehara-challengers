フォルダーの命名規則
> **<span style="color:white">例 python_1_a</span>**

* __Python__
  * ここはPythonの言語で解いていることを表します
* __1__
  * googleスプレッドシートのA列の数字になります
  * (回答のところはありません)
* __a__
  * a,mが入り、回答者の頭文字です
  * a は https://github.com/aoiorio で、
  * m は https://github.com/moriyoshi-kasuga です。
